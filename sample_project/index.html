Welcome to pyx

pyx is a toy static site generator in Python. It takes advantage of Python's with statement to intuitively express HTML documents in Python: 

with open('foo.html', 'w') as dest:
    with write_to(dest):
        with body():
            with ul():
                for text in ['foo', 'bar']:
                    with li():
                        t(text)

would translate into:

<body>
    <ul>
        <li>
            foo
        </li>
        <li>
            bar
        </li>
    </ul>
</body>
